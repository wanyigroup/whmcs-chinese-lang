### Using,使用

    上传至对应目录即可, (放 overrides 目录可覆盖源语言包)          
    upload to whmcs_install_dir

### Notes,注解

    * WHMCS 从 8.11 版本开始，WHMCS语言文件包重新生成过,后台语言包文件改为自然排序，虽然一如既往的混乱...
    * 文件 src/repare.php 是一个小工具，用于读取语言包，重排序，写入新文件，编译翻译
